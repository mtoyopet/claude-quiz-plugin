# 問題種別のフィルタリング

## タスク概要

クイズの問題種別を制限する。コードを読み解く問題（コードリーディング問題）を除外し、概念理解・コマンド理解などに絞る。

## 現状調査

- 問題生成は `src/quizGenerator.ts` の `PROMPT` で制御されている
- 現状のプロンプトに問題種別の制約は一切ない
- Claude が自由に問題を生成するため、コードリーディング問題が出ることがある

## 実装方針

`PROMPT` に問題種別の制約を追加する。

### 許可する問題種別

- **概念理解問題**: 用語の定義、概念の説明、仕組みの理解を問う
- **コマンド/構文理解問題**: CLIコマンドやAPIの使い方・オプションを問う

### 禁止する問題種別

- **コードリーディング問題**: コードスニペットの実行結果・出力を問う問題

### プロンプト追加内容（案）

```
問題の種類は以下のいずれかに限定してください：
- 概念理解問題（用語の定義、仕組み、原則の理解）
- コマンド/構文理解問題（CLIコマンド、APIの使い方、オプションの意味）

以下の種類の問題は生成しないでください：
- コードスニペットの実行結果・出力を問う問題（コードリーディング問題）
```

## 修正対象ファイル

- `src/quizGenerator.ts` — `PROMPT` テンプレートに問題種別制約を追加

## 注意事項

- プロンプトへの追記のみで対応可能。他ファイルの変更は不要。
- 指示は日本語で記述し、既存のプロンプトのスタイルに合わせる。
